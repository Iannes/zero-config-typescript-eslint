# typescript-eslint-prettier-config

This zero-config CLI is intended for react projects.This preset is not meant to kill your productivity, just worry about coding : )

## Dependencies

- prettier
- eslint
- typescript
- eslint-config-prettier
- eslint-plugin-prettier
- @typescript-eslint/parser
- @types/classnames
- @types/node
- @types/react
- @types/react-dom
- @types/react-helmet
- @typescript-eslint/eslint-plugin

- with prettier support, thanks to [eslint-plugin-prettier](prettier.io/docs/en/eslint.html#use-eslint-to-run-prettier). We take care of formatting the code for you ðŸ‘Œ

## Installation

Run the following command to start the CLI:

```sh
npx zero-config-ts-eslint

```

## Assumptions

This project assumes that your project is built with Gatsby
